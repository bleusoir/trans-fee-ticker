## Description

Transaction fee ticker for [Upbit](https://upbit.com) made with [Nest](https://github.com/nestjs/nest) framework.

## Installation

### Prerequisites
`UPBIT_API_TOKEN` environment variables. you should add environment variables.

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
